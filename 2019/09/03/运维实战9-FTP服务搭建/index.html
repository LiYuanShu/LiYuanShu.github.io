运维实战9-FTP服务搭建

  • 2019-09-03

在阿里云服务端搭建一个FTP服务，实现远程管理web服务的站点，无需再进行远程拷贝，只需在本机win10进入目录修改文件就可以马上同步了。
主要管理Apache服务的站点：

安装FTP服务：
[root@liyuanshu html]# yum install vsftpd

修改配置文件：

[root@liyuanshu ~]# vim /etc/vsftpd/vsftpd.conf

用户管理

添加一个管理用户

修改user_list白名单，只留下一个管理用户(ftp-li)即可，系统用户已经在配置文件中被禁止登陆。
[root@liyuanshu ~]# vim /etc/vsftpd/user_list

修改文件的所属者和所属组：
[root@liyuanshu ~]chown ftp-li:ftp-user -R /var/www/html/
启动服务：
[root@liyuanshu ~]systemctl start vsftpd

最后查出错误原因：密码输入错误
