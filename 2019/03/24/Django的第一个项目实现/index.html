Django的第一个项目实现

  • 2019-03-24

Django是一个开放源代码的Web应用框架。学习目的：搭建一个属于自己的网站

前期准备

虚拟环境的安装使用，可以把各个python项目分隔开来，利于管理，更合理的分配存储空间，其中选择virtualenvwrapper
vitualenvwrapper安装与使用：
①pip install virtualenvwrapper-win
②配置环境变量：在系统环境变量中新增一个(变量名：路径)————WORKON_HOME：F:\django
mkvirtualenv virtualenv_name：新建虚拟空间
lsvirtualenv：查看虚拟环境的路径和虚拟环境名
workon virtualenv_name:进入虚拟空间
deactivate：退出虚拟空间
rmvirtualenv virtualenv_name：删除虚拟空间

Django项目实现

使用某个虚拟环境：workon venv  （这个虚拟环境必须已经安装Django包）
准备一个空的文件夹并进入到这个文件夹：F:\django_project
创建一个项目：django-admin startproject projectName ，，到文件夹下查看是否创建成功
运行项目：进入到项目文件夹，执行python manage.py runserver命令运行，直接在浏览器打开http://127.0.0.1:8000/，查看是否成功
                             python manage.py runserver host:prot


结束项目运行：ctrl+C

pycharm操作

把项目导入到pycharm中，可能并不能直接运行django项目，究其原因，还是项目所处的运行环境不行。可以依据以下两个步骤正确布置运行环境
①file——》setting——》project Interpreter——》project Interpreter，在这个地方选择属于Django项目的环境

②在pycharm左上方运行图标的左边，选择edit configuration进行项目编辑

需要主要三个地方：
host：设置可以访问的IP，0.0.0.0表示允许任何人访问
prot：监听的端口号
project Inperpreter：项目的运行环境
