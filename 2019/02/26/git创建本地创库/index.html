git————本地创库

Posted on 2019-02-26

git作为一个开源的、分布式版本管理系统，同时也是linux社区下的产品。学习git，既能更深入理解linux系统，也能通过git来更好地使用github，达到以码会友的效果。

创建本地创库

1.首先在自己的电脑合适的磁盘位置创建一个空的文件夹本(test)，用来存放项目的内容
2.以test文件夹作为家目录，打开git bash here，然后初始化本地创库，命令为：git init
成功验证————>在test文件夹里产生一个 .git 文件夹。

设置标签

作用：区分不同的开发者身份。标签内容：用户名、Email地址。标签级别分为项目级别和系统用户级别

项目级别

设置用户： git config user.name name
设置email：git config user.email email
成功验证————>cat test/.git/config——————>查看到信息

系统用户级别

设置用户：git config --global user.name name
设置email：git config --global user.email email
成功验证————>cat ~/.gitconfig——————>查看到信息

添加、提交内容

查看创库状态:git status

git add filename:把文件从工作区添加到暂存区，以防文件丢失（执行后文件变为绿色）
git rm –cached filename：把文件从暂存区移除，停止追踪，工作区的原文件不会被删除
git commit filename：把暂存区的文件提交到本地库 -m：添加注释
git log：查看提交日志
–pretty=oneline——————简洁显示，每条日志信息显示为一行
git reflog：查看每个版本之间需要移动的步数
