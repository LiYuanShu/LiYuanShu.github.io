重启和关闭系统

关闭或重启系统 shutdown -r time 在time后重启系统
-h time 在time后关闭系统
关闭系统 halt -f 强制关闭系统，不调用shutdown
重启系统 reboot -f 强制重启系统，不调用shutdown
…………还有很多其他类型指令 如使用shutdown –help或者nam shutdown查看

用户管理设置

配置文件：/etc/passwd:存储系统中所有用户的相关信息，用户名、口令、UID等
/etc/shadow:为passwd的影子文件，保存了系统中所有用户和用户口令及其他在passwd没有的信息
用户组文件：/etc/group:系统所有用户组的配置信息，用户组名称、用户组GID、用户列表
/etc/gshadow

普通用户管理

注：本次的命令使用的参数少，可使用help查看具体更多的用法
添加用户： useradd userName
设置密码：password userName
更改用户密码：passed userName
修改用户名：usermod
删除用户：userdel

同理使用以上命令形式进行用户组管理:groupadd、newgrp、groupdel

第七章 磁盘分区管理

磁盘分区

一个磁盘共有四个分区:
主分区：包含操作系统启动必须的文件和数据的磁盘分区
扩展分区：存放数据和程序文件
逻辑分区：一个扩展分区可分为无数个逻辑分区

根分区：整个操作系统的根目录
Swap分区：暂时存储数据的交换分区(缓存)

IDE磁盘：
设备文件命名：/dev/hdx （x表示哪块磁盘）
分区命名：/dev/sdxy (x表示哪块磁盘，y表示分区号码)

SCSI磁盘：/dev/sdx /dev/sdxy

fdish分区管理

fdisk /dev/partitionName 进入交互模式
-b sectorsize 定义磁盘扇区大小
-l 显示磁盘分区表的信息

进入交互模式后常用命令：
查看分区：p (print) 示磁盘当前分区表
添加分区：n
修改分区类型：t
删除分区：d

LVM(logical volume manager)—逻辑卷管理

作用：操作系统对磁盘进行管理的一种机制，提高磁盘分区管理的灵活性
①物理卷(physical volume)
添加： pycreate
删除： pyremove

②卷组(volume group)
添加：vgreate groupName physicalVolume （physicalVolume参数可不要）
扩展：vgreate groupName physicalVolume
查看：gropuName

③逻辑卷(logical extents)
添加：lvcreate -L size -n logicalName groupName
修改逻辑卷大小：lvresize -L size /dev/groupName/logicalName
查看：lvdisplay /dev/groupName/logicalName
删除：lvremove /dev/groupName/logicalName

第八章 文件系统管理

文件系统管理

①创建：mkfs optionName
②标签：e2label optionName newName 方便记忆和访问
③挂载点：mkdir directoryName 创建一个文件夹，存放文件系统
④挂载：分区方式挂载：mount optionName firestoryName
标签名挂载：mount newName directoryName
⑤移除：umount directoryName

生成应用程序

将代码生成可直接执行的程序，在无安装python的环境的也能直接运行。
pyinstaller库是一个将 python 脚本转换成 Windows 上的可独立执行的可执行程序（*.exe）的工具，这样，你就可以不用装 python 而在 Windows 系统上运行这个可执行程序。

（1）安装py2exe
进入cmd。直接输入命令： pip install pyinstaller

（2）.spec文件操作
进入需要打包的程序目录下，执行 pyi-makespec FP.py（FP为代码文件名），会生成一个FP.spec 他是pyinstaller打包程序的指导书

根据需求修改.spec文件(如程序中有图片文件时)：

说明：每次因为需求不同可能需要修改不同的内容，但只要能够明白这个文件的作用，根据需求修改即可

（3）生成.exe执行文件
首先简单了解各个命令的作用
-F的含义为将原文件打包为exe文件，这里还有其他几个可选参数
-D：创建一个目录，包含exe文件，但会依赖很多文件，这是默认选项
-c：使用控制台，这也是默认选项
-w：使用窗口，无控制台
首先可以先输入命令生成一个可执行程序，然后直接把程序拖到cmd窗口执行，若有错误，根据错误信息逐步修改即可
命令：pyinstaller name.py -w (name为代码名字， 建议先使用-w，可以更容易在窗口中看到错误提示)

（4）遇到的错误：no module named “pygame”（python库名称）
错误原因： pygame库是直接在pycharm这个IDE中直接下载的，无法通过cmd加载到程序中。
解决办法： 先把pygame库复制到通过cmd pip命令下载到的文件夹中，或者直接pip命令重新下载，
能直接在cmd的python中直接import pygame,代表成功
重新执行命令：pyinstaller -F name.py –hidden-import pygame

注：跟着别人的博客笔记操作出错是正常的，只有能够理解清楚官网文档，才是解决问题的根源

matlab是大三学期《大数据挖掘》这门课需要学习的一个软件，此软件功能强大，使用也复杂多样，使用博客记录一些知识点和使用思想，以便更好地进步。

常用小知识：
分号：不显示运算结果
magic(n):生成n阶的矩阵 #魔幻
Array concatenation——cat(n,A,B):把两个矩阵拼接起来，n的取值为拼接方式。“1”为column、“2”为row，“3”为layer
reshape(A,n1,n2):改变矩阵的阶值，如可以把22阶矩阵改变为14阶矩阵。

File Access

①save workspace data to a file:
save name.mat X X为需要保存的变量名称
save name.mat -ascii X 改变编码格式，在文件中可显示
②load data stored in a file：
load（’name.mat’）

load('name.mat','ascii')

Excle File Reading

①Score=xlsread(‘Filename’,’B2:D3’) 第二个参数代表读取的范围，无第二个参数代表取钱全部数字内容
[Score Header]= xlsread(‘Filename’,’B2:D3’) Score返回数字，Header返回String
②xlswrite(‘Filename’，content，fileNumber，‘B2:D3’) 第二个参数为写入内容，第三个参数为写入到第几个表

Low-leval File I/O Function

fid=fopen(‘FileName’,’[permission]’) permission:’r’，’r+’,’W’,’W+’,’a’,’a+’
feof(fid) 判断文件是否读取完毕
fscanf(fid,’format’,size) 读取:size为尺寸可取1、2 ……
fprintf(fid,’format’,content) 写入：format:%d,%f

Drawing
