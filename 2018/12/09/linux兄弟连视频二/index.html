软件包管理

软件包分类

①源码包：可直接查看源码，自由选择安装自己想要的功能，安装过程比较困难
②二进制(rpm)包：安装较为简单

RPM软件包管理

RPM包依赖：
树形依赖
环形依赖：一次性安装
模块依赖：查询网站

安装与升级：
语法： rpm -ivh 包全名
-i(install) 安装
-v(verbode) 显示详细信息
-h(hash) 显示进度
–nodeps 不检测依赖性 

-U(upgrade)    升级

-e(erase)       卸载

-q(query)       查询某个包是否安装
-qa(all)        查询所有安装的包
-qi(information)  查询软件信息
-qp(package)      查询为安装包信息
-ql(list)         查询位置

-V(verify)      校验RPM包文件是否被修改

yum在线管理

查询：yum list
安装：yum -y install 包名
卸载：yum -y remove 包名

注：软件操作：group
