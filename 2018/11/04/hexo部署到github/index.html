hexo部署到github

  • 2018-11-04

经过在网上搜寻了很多教程，才把自己的hexo部署到了github上。好记性不如烂笔头，把自己的学习成果和经验记录下来才不会遗忘。
如何下载安装hexo请参考hexo安装

前期准备——github创库

github的强大功能在此就不说了，申请过程也非常容易。不过需要记住的重要一点：所创建的创库名称需要和自己的github名称一致，才能成功部署，否则出现404错误提示。

开始部署

登录github点击最右上角的那个图标，进入Your repositories，在点击如下图标

复制箭头的链接

把复制的内容粘贴到_config.yml文件中（repo、branch标签需要手动输入,标签和后面的内容中间留一个空格）

对url进行修改，修改结果有所偏差，结果如下：

在博客目录下运行Git Bash here输入相关命令部署：
①安装一个插件：npm install hexo-deployer-git –save
②运行命令：hexo g（作用生成相关静态网页）
③运行命令：hexo d（作用为部署到github上）
注：第一次运行hexo g命令会提示输入github账号密码等信息，安装提示进行操作即可。
我使用sublime软件对hexo进行编辑，查找相关标签ctrl+F即可。
