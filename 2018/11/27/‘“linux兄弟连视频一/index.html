权限管理命令

修改文件拥有者

名称：chown（change file ownership）
语法：chown user file(directory)

修改文件所属组

名称：chgrp（change file group ownership）
语法：chgrp group file(directory)

设置文件缺省权限

名称：umask（the user file-creation mask）
语法：umask 缺省值 （需要与或运算）
-S 显示新建文件缺省权限

文件搜索命令

名称：find
语法：find [搜索范围] fileName *任意字符 ？某个字符
-size n n表示大小，linux以数据块为单位，512k +n表示大于 -表示小于（查找大小范围内文件 -a与 -o或）
-sure sureName 根据用户查找
根据属性查找
-amin 访问时间 access
-cmin 文件属性 change
-mmin 文件内容 modify
-type 根据文件类型查找 f文件 d目录 l软链接文件

名称：locate fileName 有一个文件资料库，查找速度快（不收录temp文件夹的文件）

updatedb  更新文件资料库

语法：which commandName 查找某个命令的绝对路径

语法：grep content fileName 查找某个文件的某个内容的那一行
-i 不区分大小写
-v 排除查找

目录文件查看

用法：ls -l 显示详细信息
|grep “^name” name为d时，只显示目录，name为_时，只读文件

压缩解压命令(.gz, .tar, .tar.gz, .zip, .bz2, .tar.bz2)

一.不要目录，不保留原文件
压缩文件：gzip fileName

解压缩文件： gunzip fileName

二.可压缩目录、文件 保留原文件，压缩比不大
zip newName.zip fileName
-r 压缩目录

unzip newName.zip

三.文件很大时使用，压缩比大
bzip2
bunip2

打包或解包文件

（.tar.gz）

语法：tar -zcf NewName fileName
-c 打包
-v 显示详细信息
-f 指定文件名
-z 打包同时压缩
(.tar.bz2)
tar -cjf NewName fileName
语法：tar -xzf fileName
-x 解包
tar -xjf fileName

网络命令

发送信息：write userName
Ctrl+D:输入完毕并发送
广播信息：wall content
查看登录信息：last
lastlog 查看用户最后一次登录时间
-u 查看某个用户
显示数据包到主机间的路径：tracerroute ip地址
显示网络相关信息：netstat -l 监听
-r 路由
-n 显示ip地址和端口号
-t TCP -u UDC

配置网络：setup
